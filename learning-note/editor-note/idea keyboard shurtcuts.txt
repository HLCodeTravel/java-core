最常用快捷键

创建文件（光标在文件夹上）或插入代码（光标在代码处）
    Alt + Insert（Alt + Fn + F12）

导包
    Alt + Enter

复制代码到下一行
    Ctrl + D

查看当前文件的结构
    Ctrl + F12



-----------------------------------------------
列操作
    Alt + 左键 + 上下移动鼠标




-----------------------------------------------





-----------------------------------------------
Ctrl + Shift + Up/Down
    当前行向上/下移动一行




-----------------------------------------------
Ctrl + Shift + T
    创建当前类的测试类
