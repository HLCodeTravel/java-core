# 使用sublime text3

## 安装包
快捷键<br/>
Ctrl + ` 打开命令行<br/>
Ctrl + Shift + P 安装包的输入框<br/>
Ctrl + Shift + D 复制当前行到下一行<br/>
Ctrl + X 剪切当前行<br/>





## 关于markdown

### 安装
安装 [Markdown Editing](https://github.com/SublimeText-Markdown/MarkdownEditing) 和 [Markdown Preview](https://github.com/revolunet/sublimetext-markdown-preview) <br/>

### markdown 语法编辑

快捷键：<br/>
Ctrl + Win + V    Creates or pastes the contents of the clipboard as an inline link on selected text.<br/>

Ctrl + Win + R    Creates or pastes the contents of the clipboard as a reference link. <br/>

Shift + Win + K   Creates or pastes the contents of the clipboard as an inline image on selected text. <br/>
